SQL NOW() 函数

NOW() 函数

NOW() 函数返回当前系统的日期和时间。

SQL NOW() 语法

SELECT NOW() FROM table_name;

SQL NOW() 实例

下面的 SQL 语句从 "Websites" 表中选取 name，url，及当天日期：

实例

SELECT name, url, Now() AS date
FROM Websites;
