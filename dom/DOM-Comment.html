下面的实例使用 XML 文件 books.xml。
外部 JavaScript 用于加载 XML 文件。

createComment() - 创建一个注释节点

Comment 对象

Comment 对象表示文档中注释节点的内容。

Comment 对象属性

属性 描述
data 设置或返回此节点的文本。
length 返回此节点的文本的长度。

Comment 对象方法

方法 描述
appendData() 向节点追加数据。
deleteData() 从节点删除数据。
insertData() 向节点中插入数据。
replaceData() 替换节点中的数据。
substringData() 从节点中提取数据。
