XML Schema

XML 1.0 支持可定义文档结构的 DTD。

XML Schema 对应用程序、文档结构、属性和数据类型有着更良好的支持。

未来的 XML 版本有赖于 XML Schema 来定义 XML 文档的类型。

  • XML Schema 的结构（XML Schema Structure）规定了 XML Schema 的定义语言。
  • XML Schema 的数据类型为 XML 规定了可扩展的数据类型。
