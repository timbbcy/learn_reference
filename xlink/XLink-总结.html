XLink 概要

本教程已经向您讲解了在 XML 文档中创建超级链接的一种标准方法。

您已经明白了 XML 中的链接被分为两个部分：XLink 和 XPointer。

XLink 定义了一套在 XML 文档中创建超级链接的标准方法。XPointer 使超级链接有能力指向 XML 文档中更多具体的部分（片断）。

如需更多有关 XLink 的知识，请浏览我们的《XLink 参考手册》。
